Shortcuts
  • You can navigate this menu using keystroke shortcuts.
  • For each item having an underlined letter or number, press Alt + the underlined character to perform the item's action or expand its submenu.
  • For each active submenu item having an underlined letter or number, press either Alt + the underlined character or simply the underlined character to perform the item's action or expand its submenu.
  • If there is an active item, you can navigate the menu by pressing the arrows, or the ESC or ENTER keys, or the SPACE bar.
  • You can traverse the menu items in order by pressing the TAB key. If you're using IE6, it will work only in single window.
  • In Opera this feature is overridden by its own interface shortcuts.
